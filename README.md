### 1장. 장고 개발준비!

- 항상 **가상환경**을 만들어서 프로젝트 관리를 할 것! 여기서는 `pip`과 `venv`를 사용했지만, 나는 `conda`를 사용.
- 장고 프로젝트 생성하기 → `django-admin startproject config .`
- 로컬 서버 구동하기 → `python manage.py runserver`
